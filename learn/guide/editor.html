# VSCode 配置

使用 wxa 插件实现 .wxa 文件语法高亮~

  1. 在应用商店搜索 wxa 并安装。
  2. MacOS: command + shift + p 或 Windows: ctrl + shift + p，设置 .wxa 文件关联语言模式 wxa即可。

打开项目的.vsode/settings.json，添加以下配置：

{
    "javascript.implicitProjectConfig.experimentalDecorators": true,
    "eslint.enable": true
}

# WebStorm 配置

  1. 打开WebStorm偏好设置 Windows and Linux: File -> Settings或macOS: WebStorm -> Preferences
  2. 点击 File Types Editor -> File Types，找到 Vue.js Template，在registered Patterns中添加*.wxa

# Eslint代码校验

使用eslint校验代码，在开发阶段就解决弱*问题

        "google"
